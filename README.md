# Nero Restaurant Booking

A React & Vite frontend dApp for booking tables on an Ethereum blockchain.

This application lets guests reserve tables via MetaMask‑verified transactions, choose between two seating slots (18:00 or 21:00), and view real‑time availability. Staff can manage bookings and see table statistics through an admin interface.

<img width="1268" alt="Nero-Home" src="https://github.com/user-attachments/assets/87881583-c217-4c19-845b-7e538278c926" />

## Features

- MetaMask‑powered table reservations  
- Select date, party size and seating time  
- Real‑time availability checks  
- Admin dashboard: create, view, edit & delete bookings
- Modern, responsive UI built with React  

## Tech Stack

- **Frontend**: React.js, Vite  
- **Blockchain**: Ethereum
- **Smart Contracts**: Solidity  
- **Web3**: ethers.js
- **Development**: Hardhat
